# Product search

Use the product search tool when the customer asks about a product, price,
availability, stock, specifications, or wants help choosing an item.

Create a short search query of no more than 30 characters. Keep only the useful
product name and important specification. Do not invent product facts.

Prefer the exact component type and the specification requested by the
customer. Do not copy a long product title into the query. Words such as
`Values` describe how many options a product has; do not translate them into
`set` or `kit`. For example, a request for a 24 kOm, 0.25W resistor should use a
query such as `resistor 24k 0.25W`.

If the request is too vague to make a useful search, ask one short clarifying
question instead of searching random products.
